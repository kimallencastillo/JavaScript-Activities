What is Decimal Numbering System

A numbering system that uses 2 symbols in representing quantities
Symbols are 0 and 1
The base for this numbering system is 2
Binary is the language of computers. Information is sent to and from the processor in the binary
system which consists of only two digits (1 and 0)
Example: 11110, 101110, 101, 110101
