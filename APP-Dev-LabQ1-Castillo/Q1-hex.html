What is Hexadecimal Numbering System ?
  • A numbering system that uses 16 symbols in representing quantities
  • Symbols are the numbers 0 to 9, and the letters A to F
  • The base for this numbering system is 16
  • Hexadecimal numbers are used to represent address of memory locations
  • Example: ABC, 11CD,FACE5, 9015D
